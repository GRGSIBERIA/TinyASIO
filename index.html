TinyASIO

ASIO SDKのラッパーライブラリ

Download this project as a .zip file Download this project as a tar.gz file

概要

TinyASIOはASIO SDKのラッパーライブラリです． ASIO SDKでの開発を省力化するために作られました． 音声入力を音声出力に流すだけのコードがたったの5行で書くことができます．

TinyASIOは簡単に書けるだけでなく，簡単に拡張することができます．
